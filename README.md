# Hi, mom!

A blazingly fast JavaScript library to say hi to your mom!

## Install

#### yarn

```bash
yarn add hi-mom
```

#### npm

```bash
npm i hi-mom
```

## Usage

#### CommonJS

```javascript
const { hiMom } = require("hi-mom");

console.log(hiMom()); // Hi, mom!
```

#### ES Modules

```javascript
import { hiMom } from "hi-mom";

console.log(hiMom()); // Hi, mom!
```
